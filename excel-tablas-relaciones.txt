Descripción:
Este proyecto muestra cómo transformar y analizar datos usando tablas relacionadas en Excel. Incluye ejemplos de consolidación, estructura de relaciones, y ejercicios prácticos para dominar el análisis de datos relacional en hojas de cálculo. Ideal para quienes buscan llevar el análisis de datos en Excel a un nivel profesional.
